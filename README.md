# 🐾 PetCare Website

PetCare is a responsive pet healthcare website developed using **HTML, CSS, and JavaScript**. The platform provides an easy and user-friendly experience for pet owners to explore healthcare services, search for doctors, book appointments, and view customer reviews.

## ✨ Features

* 🩺 Doctor search functionality
* 📅 Online appointment booking
* ⭐ Reviews and ratings section
* 📱 Responsive and modern UI
* 🔍 Easy navigation and clean design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 Project Purpose

The main goal of this project is to create a simple and modern pet care platform that improves accessibility to pet healthcare services online.
